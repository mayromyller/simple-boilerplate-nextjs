# Boilerplate NextJS Project

## A simple boilerplate for projects using Next, React and Styled Components + eslint configuration, prettier, editorconfig, babel and SSR with Styled Components

### For testing

`yarn dev`

### For build

`yarn build`

### Check lint

`yarn lint`
